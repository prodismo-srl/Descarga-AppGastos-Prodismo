Descripción

Aplicación para el control y gestión de gastos de viajes de Prodismo SRL. Optimiza el proceso de reporte y aprobación de gastos corporativos de Prodismo SRL

1. Introducción

Proyecto: Aplicación Web de Control de Viajes en Plataforma para anidar futuras App Web.

Dominio: @prodismo.com

2. Características Generales

Características

  • Sitio web moderno, adaptable a cualquier resolución y visible en cualquier tipo de dispositivo
  • Modelado y desarrollo de Base de Datos
  • Desarrollo y diseño de la aplicación
  • Gestión de usuarios y permisos
  • Reportes para la Gestión Administrativa
  • Dashboards interactivo de Administradores y Usuarios
  • Sistema multi-moneda
  • Carga de gastos offline

Módulos de la Aplicación Web

  • Dashboards
  • Administración de Usuarios, Roles y Permisos
  • Administración de Viajes
  • Ingreso de Gastos
  • Gastos Compartidos e invitaciones
  • Parametrización
    • Tipo de Gatos
    • Monedas
  • Reportes
